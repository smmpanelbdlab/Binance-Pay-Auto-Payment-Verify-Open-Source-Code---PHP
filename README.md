## 📖 How to Use

1. Enter your license key and amount.
2. Click **Pay Now** – the modal opens with Binance ID & QR.
3. Send the exact amount via Binance Pay.
4. Paste the Order ID and click **Verify**.
5. Instant success/failure feedback.

## 🛠️ Installation

1. Clone this repository.
2. Place the `binance-pay-demo.php` file on your PHP server.
3. Ensure `cURL` is enabled.
4. Open the file in a browser and start testing.
